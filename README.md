# Quick Start

## Dependencies
- Python (v3.8 or higher)
- Pygame (v2.0 or higher)

## Setup

1. Create a Python virtual environment:
```
python3 -m venv venv
```

2. Install dependencies:
```bash
pip install pygame
```

3. Run the game:
```bash
python3 main.py
```
